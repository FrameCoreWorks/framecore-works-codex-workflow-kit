# Code Of Conduct

This project expects practical, respectful collaboration.

## Expected Behavior

- Stay focused on improving the workflow kit.
- Give clear technical feedback with concrete examples.
- Respect privacy and do not request or expose private project context.
- Avoid hostile, discriminatory, harassing, or disruptive behavior.
- Assume public repository artifacts may be read by people with different skill levels.

## Unacceptable Behavior

- Publishing secrets, credentials, private URLs, local paths, or private project material.
- Harassment, personal attacks, threats, or discriminatory language.
- Repeated off-topic disruption.
- Attempts to weaken installer safety, privacy checks, or user consent boundaries without clear justification.

## Reporting

Use GitHub moderation and repository reporting tools when available. For security-sensitive issues, follow `SECURITY.md` and do not publish sensitive details publicly.

Maintainers may remove comments, close issues, request edits, or restrict participation when behavior harms the project or its users.
