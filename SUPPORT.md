# Support

FrameCore Works Skill Kit is maintained as a GitHub-first workflow kit.

Use GitHub issues for:

- installation problems
- onboarding questions
- documentation gaps
- validation or privacy-audit failures
- safe install, repair, update, or uninstall behavior
- feature requests for provider-neutral workflow improvements

## What To Include

Include only sanitized diagnostics:

- kit version, release tag, or commit SHA
- operating system
- Node.js version
- install mode: guided, project-local, global, update, repair, or uninstall
- command that failed, with private paths replaced by placeholders
- expected behavior
- actual behavior
- sanitized output or finding code
- whether `.framecore/manifest.json` exists in the target workspace

Do not post secrets, credentials, private URLs, private project context, generated confidential outputs, or local machine paths in support requests.

For security-sensitive reports, follow `SECURITY.md` instead of opening a detailed public issue.
